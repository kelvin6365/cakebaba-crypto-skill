# Cross-Asset Correlation & Market Context — Cakebaba 跨市場分析

## 🌐 點解需要跨市場分析

加密貨幣**唔係孤島**。Cakebaba 嘅分析永遠包含:

1. **美股(尤其 SPY、QQQ)** — 影響 BTC 嘅 risk-on/risk-off sentiment
2. **黃金 (Gold / XAU)** — 反向 indicator,risk-off 嘅資金流向
3. **DXY(美元指數)** — DXY 升通常 BTC 跌
4. **ETH/BTC ratio** — 反映 altcoin 季節嘅 leading indicator
5. **BTC dominance** — BTC vs 其他山寨嘅資金流動

唔睇呢啲,你嘅 BTC 分析等於用一隻眼睇市場。

## 📊 主要相關性對照表

### A. SPY / 美股 ←→ BTC

| SPY 狀態 | BTC 預期反應 | 操作意涵 |
|---------|------------|---------|
| **SPY 強勢上升 + 創新高** | BTC 通常跟升(risk-on) | 做多 BTC 有利,可放鬆 |
| **SPY 喺 ATH 附近震盪** | BTC 偏弱,容易 consolidation | 縮小倉位,等明朗 |
| **SPY 跌穿關鍵支撐** | BTC 大概率跟跌 | **唔好做多 BTC**,小心做空機會 |
| **SPY 跌幅 < BTC 跌幅** | BTC 弱於美股,內部問題 | 警惕 BTC 嘅 setup 失效 |
| **SPY 升 BTC 跌** | BTC 內部 weakness,divergence | 等 BTC 自己 structure 修復 |

**Cakebaba 觀點(片中):**
- SPY 已經喺 phase top(可能單頂或雙頂)
- 喺 box 入面 consolidation
- 18% rally 之後 6-7% retracement 係正常
- 未到 0.618 / 0.5 Fibonacci 支撐
- **意涵:SPY 唔啱即刻買,等 retracement 完成 → 同樣意味 BTC 短期難有突破**

### B. Gold ←→ BTC

| Gold 狀態 | BTC 反應 | 解讀 |
|----------|---------|------|
| **Gold 強勢上升** | BTC 短期可能跟升(都係 alternative asset) | 但長期 Gold 強 = risk-off,BTC 終須跟跌 |
| **Gold 橫盤** | BTC 自主行情 | 跟 BTC 自己 structure |
| **Gold 暴跌** | risk-on 環境,BTC 通常受益 | 做多 BTC 有利 |

**Cakebaba 觀點(片中):**
- Gold 而家冇明確趨勢,喺 channel 入面 oscillate
- Day trader 可以喺 channel 上下軌做反向
- 突破上軌嘅 trendline 先係強勢 bullish

### C. DXY(美元指數)←→ BTC

**反向關係(基本上是負相關):**
- DXY 升 → 美元強 → 風險資產(包括 BTC)跌
- DXY 跌 → 美元弱 → BTC 升

**關鍵 DXY level:**
- DXY > 105 = 強美元期,BTC 通常壓抑
- DXY < 100 = 弱美元期,BTC 通常有利
- DXY 短期 4-5% 急升 = BTC 大概率 -10%+ correction

### D. **ETH/BTC Ratio** — 山寨季嘅 Leading Indicator

呢個係 Cakebaba 特別強調嘅 indicator。

**ETH/BTC ratio = ETH 嘅 USD 價 / BTC 嘅 USD 價**

| ETH/BTC 趨勢 | 意涵 | 對山寨季嘅意義 |
|-------------|------|--------------|
| **上升中** | ETH 強於 BTC | **山寨季可能即將開始** |
| **下跌中** | ETH 弱於 BTC | **BTC 主導,山寨 underperform** |
| **創新低** | ETH 極度 underperform | 山寨季肯定未開始,可能仲喺底部 |
| **底部反轉** | 山寨開始輪動 | 啱啱開始嘅階段,睇 follow-through |

**Cakebaba 觀點(片中):**
- ETH/BTC ratio 已經 retrace 到 0.618 Fibonacci
- 技術上呢個唔似 ETH 嘅底部
- **預期 ETH 仲會繼續 underperform BTC**
- 操作意涵:**唔好提早入山寨多倉**,等 ETH/BTC 底部出現先做山寨

**👉 點樣用 ETH/BTC ratio 指導交易:**

1. **ETH/BTC ratio 下跌中:**
   - 做多 BTC 優於做多 ETH 或山寨
   - 山寨做空可考慮(順 BTC dominance 升勢)

2. **ETH/BTC ratio 橫盤:**
   - BTC 同山寨表現接近
   - 做多選 setup 最強嘅嗰個

3. **ETH/BTC ratio 上升中:**
   - 做多 ETH 或山寨優於 BTC
   - **山寨季可能展開**

## 🗓️ 美股假期 & 週末效應(Cakebaba 重點)

呢個係好實用嘅 trader 知識:

### 美股交易日 (Mon-Fri, 9:30am-4pm ET)
- BTC 流動性最高
- BTC 跟美股聯動最強
- 大型 move 通常喺呢段時間發生
- 新聞、宏觀數據都喺呢段時間發布

### 美股盤後 (4pm-9:30am ET 隔日)
- 流動性下降,但仲算 OK
- 亞洲時段(美股盤後)有 Asian liquidity 接住
- 可以有獨立行情

### **週末 (Sat-Sun)** — **危險時段**
- **流動性最低** — bid-ask spread 闊
- **容易插針** — 鯨魚/做市商更易推動價格
- **Liquidation hunting 高發** — sweep clusters 容易喺呢度發生
- **新聞驚喜風險** — 監管/事件喺週末發布(無法即時 react)
- **操作建議:**
  - 縮細倉位 30-50%
  - 止損放寬少少(避免 wick 掃止損)
  - 100x 絕對唔好(波動太大,無法控制)

### **美股公眾假期** — **更危險**
- 機構同 retail 都休市
- BTC 流動性類似週末
- **Cakebaba 例子(片中):週一美股假期 → BTC 預期 3 日 consolidation**
- 假期前後最易出現 unexpected moves
- **操作建議:**
  - 假期前停止新開倉
  - 持倉收緊止損
  - 假期後第一日先睇方向再決定

### 重大宏觀事件日
- FOMC 議息(每年 8 次)
- CPI / PCE 公佈日(每月)
- NFP 非農就業(每月第一個週五)
- **24h 內唔開新倉**,持倉縮細

## 🎯 跨市場分析嘅完整流程(每次 BTC 分析前必做)

```
Step 0(跨市場 context 檢查,15 分鐘):

1. 開 SPY 4h 圖
   ├─ 趨勢方向?
   ├─ 接近 S/R?
   └─ 有冇 setup?

2. 開 Gold 日線圖
   ├─ Risk-on 定 risk-off?
   ├─ 突破 / 跌穿?
   └─ 對 BTC 有冇 implication?

3. 開 DXY 日線圖
   ├─ 強美元 vs 弱美元期?
   ├─ 接近 100 / 105 關鍵 level?
   └─ 短期趨勢方向?

4. 開 ETH/BTC ratio 4h
   ├─ 上升 vs 下跌?
   ├─ 接近 Fibonacci?
   └─ 山寨季 implication?

5. 睇日曆
   ├─ 24h 內有大事件?
   ├─ 美股假期?
   └─ 週末效應?

→ 綜合判斷:BTC 嘅外部 context 係 favorable / neutral / hostile?

如果 hostile → 縮小倉位或 WAIT
如果 favorable + BTC 自己 setup OK → 可入場,可考慮高槓桿
```

## ⚠️ 跨市場分析嘅警告

1. **相關性唔係 100%** — BTC 有時可以同 SPY decouple(尤其加密本身有大新聞)
2. **時間滯後** — SPY 嘅 move 可能要 1-2 日先影響到 BTC,唔係即時
3. **單獨 indicator 唔可靠** — 一個 cross-asset signal 唔夠,要多個 align
4. **市場 regime 會變** — 2017 加密同股票相關性低,2020+ 大幅提高。要留意制度變化

> **BTC 從來唔係單獨 trade 嘅。**

Cakebaba 嘅實戰分析永遠 cover BTC + ETH + SPY + Gold,呢個唔係炫技,係**必要嘅 context**。
